### Installation

Github installtation
```
git clone 
cd microcat
# Make all python scripts to be executable
chmod +x microcat scripts/*.py 
# Add microcat to the PATH, so it can be called elsewhere in a terminal
PATH=`pwd`:$PATH 
```
